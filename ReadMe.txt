需安装模块:
    paramiko,pymysql,sqlalchemy,prettytable

使用帮助:
    1.数据库:需要自己插入一些数据,使用前需要修改数据的的IP地址的用户名密码,
        修改位置在db/db_strut中"engine = create_engine..."
    2.环境设置:可以在linux系统中创建某个用户,在/home/username/.bashrc添加以下两行:
        /usr/bin/python3 /home/chengc/Day13/homework/bin/js.py
        logout
    3.用户:创建的用户,需要在数据库userprofile表中有记录,并且必须与数据库中有相关操作权限的服务器信息匹配,
        为了显示出效果来,建议多匹配几条服务器洗洗
    4.批量上传和批量命令行比较简单,直接输入序列号后面跟着命令或者文件以及路径即可
